Note For Theme Developers:  The following is a list of all the button
types available in the default xml files.  Each menu item has been given
its own buttontype to allow theme makers to have greater flexibility
when creating icons for their theme.  If you do not have an individual
icon for a particular menu item in your theme, you may choose to make an
entry in your theme.xml that links to a duplicate icon picture in your
theme.
