GetSearchCapabilities - Returns the searching capabilities that are supported by the device.
Params: none    
 
GetSortCapabilities - Returns the CSV list of meta-data tags that can be used in sortCriteria.
Params: none    
 
GetSystemUpdateID - Returns the current value of state variable SystemUpdateID.
Params: none    
Can be used by clients that want to ‘poll’ for any changes in the Content Directory (as opposed to subscribing to events).
